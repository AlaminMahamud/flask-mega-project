# Flask Mega Project

Codebase for Flask Examples.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites
1. python3
2. pip3
3. virtualenvwrapper

Install `python3`
```bash
sudo apt-get install -y python3
```
Install `pip3`
```bash
sudo apt-get install -y python3-pip
```

Install `virtualenvwrapper`
```bash
sudo pip3 install vitualenvwrapper
```

On your `.zshrc` or `.bashrc`
```bash
export WORKON_HOME=$HOME/.virtualenvs
export PROJECT_HOME=$HOME/projects
source /usr/local/bin/virtualenvwrapper.sh
```

```bash
source ~/.bashrc
#or
source ~/.zshrc
```

### Installing

Create a virtualenv

```bash
mkvirtualenv flask-mega-project
workon flask-mega-project
```

Clone the Repo

```bash
git clone https://github.com/AlaminMahamud/flask-mega-project.git
cd flask-mega-project
```

Install the requirements

```bash
pip install -r requirements.txt
```

## Running the tests

```bash
python -m unittest discover -v
```
